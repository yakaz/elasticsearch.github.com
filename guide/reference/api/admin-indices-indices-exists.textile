p. Used to check if the index (indices) exists or not. For example:

<pre class="prettyprint">
curl -XHEAD 'http://localhost:9200/twitter'
</pre>

p. The HTTP status code indicates if it exists or not. A @404@ means its not found, and @200@ means its there.
